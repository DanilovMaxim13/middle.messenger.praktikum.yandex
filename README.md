## Установка и запуск
Склонируйте репозиторий
`git clone https://github.com/DanilovMaxim13/middle.messenger.praktikum.yandex.git`

Установите зависимости 
`npm install`

Запуск сборки для разработки 
`npm run dev`

Запуск сборки для продакшена 
`npm run start`
